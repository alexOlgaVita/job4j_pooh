# job4j_pooh

### О проекте
Приложение реализует работу брокера для асинхронного обмена сообщениями между поставщиком и потребителем сообщений.

### Стек технологий
Java 17, JUnit 5

### Требования к окружению
Java 17, Maven 3.4.0

### Запуск проекта

``` mvn clean install  ```
и далее запускаем приложение (класс PoohServer)
